## Objective 
how to dump the hexadecimal for a binary with offsets

## Status 
Completed

## Files 

- **main.cpp** : Reads a file into 1kb chunks and prints it out.
- **Makefile** : compiles into a single executable named hdmp.
- **output.txt** : contains the hexdump of a file. 
## How to run
```bash
make
hdmp <input_file_path>

```
